# SimulateLocation
一个让真机改变定位的Demo

介绍：一个可以真实改变手机定位的Demo

工具：谷歌地图 http://www.google.cn/maps

步骤：

1.在谷歌地图中找到自己想要模拟的位置


2.拿到坐标


3.在GPX文件中修改对应坐标，按照在谷歌中拿到的前后顺序放进对应的位置即可


4.然后在真机上运行，你就会发现手机的定位改变了，无论是打开高德，还是微信钉钉，你的定位都改变了！


坐标系知识：

iOS原生坐标系为 WGS-84

高德以及国内坐标系：GCS-02

百度的偏移坐标系：BD-09

GPX文件中需要的坐标系  WGS-84
